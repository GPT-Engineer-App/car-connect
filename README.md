# car-connect

Build a cross platform application called shifthappens designed for car enthusiasts to connect, organize events, create car clubs, and join existing clubs. It provides a platform for enthusiasts to share their passion for cars, post pictures, and engage with like-minded individuals.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/car-connect.git
cd car-connect
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
